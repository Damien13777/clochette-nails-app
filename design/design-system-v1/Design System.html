02

Typographie

Aa

Serif · Titres

CINZEL

Display serif romain, esprit Aesop / Bulgari / Tiffany. Réservé aux titres en capitales. Pas d'italique natif — bascule automatiquement sur Inria Serif italic pour les emphases.

--font-serif · Cinzel · 400 / 500 / 600 / 700 / 800 / 900 (pas d'italique)

Aa

Serif body · Light

Inria Serif

Serif aérien et lisible en graisse Light 300. Apporte une chaleur éditoriale aux paragraphes et formulaires — sans concurrencer Cormorant en titre.

--font-sans · Inria Serif · 300 / 400 / 700 + italic

Display caps · accents premium

Display

JULIUS SANS ONE

Réservé aux capitales : eyebrows, badges, libellés de formulaire, boutons. Donne le caractère premium sans alourdir la lecture longue.

--font-display · 400

Sans UI · dense

UI

Manrope

Réservé aux contextes UI denses : valeurs d'inputs, helper text, messages d'erreur, tables, toasts, admin. Plus neutre que l'Inria Serif, lisible à petite taille.

--font-ui · Manrope · 300 / 400 / 500 / 600 / 700

14

Principes UI

Quelques règles d'usage qui priment sur tout le reste — pour préserver la douceur, la lisibilité et la sobriété de la marque.

01

Mobile-first

Concevoir pour un écran 375 px d'abord. Le desktop n'est qu'une amplification.

02

WCAG AA strict

Contraste minimum 4.5:1 sur le texte. Tous les états interactifs visibles au clavier.

03

Pas d'emoji, pas de néon

Iconographie Lucide stroke 1.5. Aucun effet criard, paillette ou dégradé saturé.

04

Micro-interactions subtiles

Toutes les transitions sont de 0.3s, en easing doux. Jamais de bounce ou de rebond.

05

Respiration généreuse

Whitespace prioritaire. Préférer une seule colonne aérée à deux colonnes serrées.

06

Tokens, jamais de hex

Toute couleur, taille ou ombre passe par une variable --*.
